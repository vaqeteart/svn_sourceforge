本程序的作用是使用嵌套类

一.图文件的格式例如：
4
0 1 0
0 3 0
1 0 0
1 3 0
2 3 0
3 0 0
3 1 0
3 2 0
其中，
4表示图中的点的总数
剩下的,
前两列表示图中的边
最后一列表示边的代价


二.关于边权值函数的说明：
权值函数默认为分段函数.时间t的模为15000
有关边的权值函数可以自己定义.
因为边的权值函数随着网络的不同而不同，不容易编写其输入函数的通用输入形式，所以如果要想改变边的权值函数的话，那就自己根据适合的网络修改权值函数MyEdge::getCost(int i,int j,int t)
邻接矩阵
